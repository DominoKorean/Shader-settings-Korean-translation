# 셰이더 설정 한글번역 프로젝트
마인크래프트 셰이더의 설정을 한국어로 번역해 
한국인 누구나 쉽게 설정을 만질 수 있는것을 목표로 합니다.

최대한 많은 셰이더를 번역하는것을 목표로 하고있습니다.

## 적용법
zip의 압축을 풀지 마시고
압축 상태로 편집이 가능한 프로그램을 이용해
셰이더 파일 내에 lang폴더에
ko_KR.lang 파일을 집어넣으시면됩니다.
